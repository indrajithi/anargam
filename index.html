Welcome to Anargam

This is a simple python program that let you find the anargam or solve jumbled words.

How to use it

The code is written for Python 2.7.10 so make sure you use that python --version.

git clone https://github.com/indrajithi/anargam.git

cd anargam/

python anargam.py
